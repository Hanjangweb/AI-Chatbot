# AI Chatbot (RAG System)

This is a full-stack MERN application that uses Retrieval-Augmented Generation (RAG) to provide intelligent chat responses.

##  Features
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **AI Logic:** RAG implementation for context-aware answers.
- **Deployment:** Optimized for Vercel.

## Installation & Setup

### 1. Clone the repository
```bash
git clone [https://github.com/Hanjangweb/AI-Chatbot.git](https://github.com/Hanjangweb/AI-Chatbot.git)
cd AI-Chatbot

#2. Backend Setup
cd backend
npm install
# Create a .env file and add your API keys
npm start

#3. Frontend Setup
cd ../frontend
npm install
npm run dev
